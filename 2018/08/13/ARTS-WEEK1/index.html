Algorithm

LeetCode: https://leetcode.com/problems/two-sum/description/

这个题目一开始的实现思路就是2层嵌套循环， 第一层循环是必须的遍历过程， 但第二层循环其实可以看做是在一堆数据里查找1个确切的数字， 这个很容易想到可以用hash（python里边可以用字典）。

这个题目还容易忽略的一点儿是， 遍历的数字自身相加也可能得到目标数字， 造成结果是两个相同的下标。

Review

Docker Review

https://docs.docker.com/engine/docker-overview/

Docker平台

  1. Docker是轻量级的， 因为它直接运行在宿主机的内核上， 并不会想KVM那样额外加载Hypervisor， 这也就意味着在给定的硬件上可以运行字的容器比虚拟机更多
  2. Docker可以运行在宿主机，也可以运行在虚拟机内

    Docker引擎

    Docker引擎是一个C/S结构的应用程序，包含以下组件：
  3. server，一个常驻后台的daemon进程(dockerd)
  4. REST API， 让应用程序可以跟daemon进程交互的接口
  5. docker命令， 一个客户端命令行工具

Docker Daemon负责创建和管理Docker对象， 如images、containers、networks、volumes

我们可以用Docker做什么

Docker简化了软件开发生命周期，它使开发者可以在本地运行服务的容器中使用标准环境。 容器非常适合持续集成和持续交付的工作流。
例如以下应用场景：

  • 开发者在本地编写程序，并通过Docker容器分享给同事
  • 用Docker容器将他们的应用程序推送到测试环境，并进行测试
  • 当开发者发现bug时， 他们在开发环境修复bug并重新部署它们到测试环境进行测试和验证
  • 当测试完成后，直接将新image推送到生产环境
