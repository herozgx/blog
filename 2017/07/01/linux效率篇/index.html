记录一些高效的Linux操作

linux下的操作尽量远离鼠标， 其实其中的操作无外乎两种：

  • 文本编辑器操作
  • 命令行操作

所以掌握命令行和文本编辑器的快捷键即可在Linux里边行云流水般生存。

文本编辑器主要由两个： Vim 和 Emacs, 前者号称编辑器之神， 后者号称神的编辑器， 在江湖中各有拥戴者万千， 形成两大门派。 如果你采用的是Emacs编辑器， 那么恭喜你， 你已经掌握了命令行中的几乎所有快捷键操作， 将linux下边的两种操作合二为一了。 如果你用的是VIM编辑器，那么除了vim中的各种快捷键，你还需要掌握常用的命令行快捷键。
