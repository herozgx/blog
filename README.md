### 下载项目到本地

```bash
git clone https://github.com/herozgx/blog.git
```

### 安装node模块
```
npm install    #根据目录中的package.json
```

### 创建新post

``` bash
$ hexo new "My New Post"
```

### 生成静态文件

``` bash
$ hexo generate
```

### 运行本地服务器

``` bash
$ hexo server
```

### 将静态文件推送到gh-pages分支

``` bash
$ hexo deploy
```
